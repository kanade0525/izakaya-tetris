# 居酒屋テトリス

居酒屋で飲み食いしながら遊ぶ、ターン制テトリスパズルゲーム。

> 出典: [居酒屋テトリス - YouTube](https://www.youtube.com/watch?v=9S2_bQ4ICKIこれ)

## 遊び方

### 1. まずゲームオーバーになるまで遊ぶ

普通のテトリスをプレイしてゲームオーバーにします。ゲームオーバーになった盤面がそのまま居酒屋テトリスのスタート地点になります。

（ランダムな盤面からいきなり開始することもできます）

### 2. 居酒屋で飲み食いするたびにストックを貯める

居酒屋で一品食べたり一杯飲んだりするたびに、画面上部の「+」ボタンでストックを1つ追加します。これは現実世界での操作です。

### 3. ストックを消費してブロックを落とす

ストックを1つ消費して「ブロックを落とす」を実行すると、9種類の中からランダムで1つが抽選されます。

| 種類 | 内容 |
|------|------|
| 通常ブロック (7種) | I, O, T, S, Z, J, L のいずれか |
| 何でもブロック | 好きなブロックを1つ選べる |
| ハズレ | 何も起きない |

通常ブロック・何でもブロックが出たら、3秒のカウントダウン後にブロックが落ちてきます。操作して好きな場所に置きましょう。

### 4. 出てほしくないブロックを除外できる

追加でストックを消費することで、抽選から特定のブロックやハズレを除外できます。1つ除外するごとに+1ストック消費（例: 2つ除外なら合計3ストック）。

### 5. 最終段を消したらクリア!

盤面の一番下の行のブロックをすべて消せばクリアです。全消しする必要はありません。

## その他の機能

- **Undo / Redo** - ヘッダーのボタンまたは Ctrl+Z / Ctrl+Y で操作を戻せます
- **オートセーブ** - 居酒屋モードの状態はlocalStorageに自動保存。ブラウザを閉じても「続きから」で再開できます
- **ライン消去演出** - 行が揃うとヒットストップ（フラッシュ + 画面振動）が発生します

## 技術スタック
