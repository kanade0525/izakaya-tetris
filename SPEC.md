# 居酒屋テトリス 仕様書

## 概要

居酒屋で飲食するたびにストックが貯まり、ストックを消費してブロックを1個ずつ落とし、最終段を消すことを目指すターン制パズルゲーム。通常のテトリスモードも搭載。

---

## モード

### クラシックモード
- 通常のテトリス
- 自動落下、レベルアップで加速
- ゲームオーバー後、居酒屋モードへ移行可能（盤面引き継ぎ）

### 居酒屋モード
- ターン制。自動落下なし
- ゴール: 最終段（一番下の行）を消す
- 開始方法:
  - タイトルから「居酒屋モード」を選択（ランダム盤面）
  - クラシックモードのゲームオーバー盤面から移行

---

## 居酒屋モード 詳細仕様

### ストックシステム
- 居酒屋で一品食べる / 一杯飲むたびにストックを+1する（現実世界での操作）
- ヘッダーにストック数と[+][-]ボタンを表示
- ストックが0の場合、「ブロックを落とす」は実行不可

### ブロックを落とすフロー

```
[ブロックを落とす] ボタン押下
  ↓
確認ダイアログ「ストックを1消費してブロックを落としますか？」
  ├→ 除外設定（任意）
  ├→ 落とす
  └→ キャンセル
  ↓
抽選（9種からランダム）
  ├→ 通常ブロック → 3秒カウントダウン → 操作 → 固定 → 停止
  ├→ ワイルドカード → ブロック選択 → 3秒カウントダウン → 操作 → 固定 → 停止
  └→ ハズレ → 「ハズレ」表示 → idle
  ↓
ストック消費（1 + 除外数）
```

### 抽選プール（9種・等確率）

| # | 種類 | 説明 |
|---|------|------|
| 0 | I | 水色・棒型 |
| 1 | O | 黄色・四角 |
| 2 | T | 紫・T字 |
| 3 | S | 緑・S字 |
| 4 | Z | 赤・Z字 |
| 5 | J | 青・J字 |
| 6 | L | オレンジ・L字 |
| 7 | ワイルドカード | 好きなブロック1つ選べる |
| 8 | ハズレ | 何も起きない |

### 除外システム
- 「ブロックを落とす」実行時に、追加でストックを消費して出てほしくないブロックを除外できる
- 1つ除外につき+1ストック（例: 2つ除外なら合計3ストック消費）
- ハズレも除外対象に含む
- 使わないことも可能（選択式）

### 操作中の挙動
- 通常ブロック / ワイルドカード選択後: 3秒のカウントダウン後にブロック落下開始
- 落下速度は遅め（1.5秒間隔）
- 左右移動・回転・ハードドロップの操作可能
- ブロック固定後、盤面はそのまま停止（次の「ブロックを落とす」まで待機）

### ライン消去
- クラシックと同じヒットストップ演出（フラッシュ + 画面振動）
- 消去後に最終段クリア判定

### 盤面
- 幅: 10マス（固定）
- 初期盤面: 10x25（上5行は空、下20行はランダムにブロックが埋まった状態）
- ブロックが積み上がって上部に余裕がなくなったら上方向に拡張（最大40行）
- 下から20行目に赤い基準線を表示
- カメラがスクロールして盤面を追従

### 勝利条件
- 最終段（一番下の行）のブロックがすべて消えたらクリア

## 技術スタック

- Nuxt 3 (SSR: false, SPA)
- Canvas API
- TypeScript
- AWS Amplify (ホスティング)

## ファイル構成

```
composables/
  useTetris.ts          — 共通ゲームロジック（盤面・移動・回転・消去）
  useIzakayaMode.ts     — 居酒屋モード専用（ストック・抽選・除外・ステートマシン・永続化・undo/redo）
pages/
  index.vue             — Canvas描画 + 全UIオーバーレイ
assets/css/
  main.css              — 全スタイル
```
